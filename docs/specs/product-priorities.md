# Приоритеты продукта NeuroSync Pro

Зафиксировано по согласованию с автором проекта.

## База пользовательской ценности

**Медитация и концентрация** — центральный сценарий. Всё остальное (формулы, тонкая настройка параметров, расширенная аналитика) поддерживает этот фокус, а не перетягивает интерфейс на «лабораторный режим» ради самого режима.

## NeuroExperimenter → что переносить в первую очередь

- **Графики для наблюдения** — непрерывная визуализация ЭЭГ/спектра в реальном времени, чтобы пользователь видел динамику.
- **Ручная отладка** — возможность пользователю самому смотреть на сигнал, менять масштаб/каналы, отмечать моменты; без этого сложно доверять автоматике и исследованиям.

Формулы, выбор полос и параметров обработки — **важны**, но вторичны относительно ясного наблюдения и сценариев «медитация / концентрация».

## Sweep Tone Generator (Tone Generator PRO) — направление развития

По опыту использования: генератор задаёт высокую планку — **желательно со временем покрыть или приблизиться ко всем предусмотренным там функциям**, с **оптимизацией под агентов**:

- явные API/события для смены режима (тон, sweep, шумы, многоканальность, пресеты и т.д.);
- предсказуемые параметры и лимиты безопасности (громкость, скорость развёртки);
- логирование команд агента и состояния генератора для воспроизводимости экспериментов.

Полный паритет с мобильным приложением может идти **поэтапно**: сначала ядро (тон + sweep + запись + связь с ЭЭГ и базовыми пресетами), затем расширение к возможностям эталона.

## Краткая формула приоритетов

1. Медитация и концентрация (сценарии и UX).  
2. Графики и ручной контроль ЭЭГ.  
3. Мощный аудиомодуль по мотивам Sweep Tone Generator + удобство для AI-агентов.  
4. Формулы и «лабораторные» параметры — там, где усиливают пункты 1–3.
